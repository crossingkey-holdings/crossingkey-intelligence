# Provenance and publication standard

## Scope

This repository documents selected CrossingKey work and perspective developed from June 2025 through July 21, 2026, with future updates recorded as discrete releases.

## Evidence standard

Published material should distinguish among:

- **Authored** — created directly by CrossingKey.
- **Developed** — implemented or materially advanced by CrossingKey.
- **Curated** — selected and organized from external or prior material.
- **Planned** — a stated direction, not a delivered capability.
- **Archived** — preserved historical material that may not represent current practice.

## Publication boundary

Public documentation is intentionally descriptive rather than exhaustive. It should establish context and provenance without exposing confidential methods, credentials, customer information, private source code, security controls, or implementation details that would create avoidable risk.
